# CS30064-NeuralNetwork-DeepLearning

复旦大学 CS30064《Neural Network and Deep Learning》课程代码仓库。

本仓库用于集中管理课程各次实验/项目代码、报告与结果文件。

## 仓库结构

- `PJ1/` — Project 1: 基于 NumPy 从零实现 MLP 与 CNN，完成 MNIST 分类、优化与正则化实验、鲁棒性分析和错误分析（具体依赖、运行方式见 `PJ1/README.md`）。

> 后续项目可按同样结构继续添加，如 `PJ2/`、`PJ3/` 等。

## 快速导航

- PJ1 代码与说明：`PJ1/README.md`
- PJ1 报告源文件：`PJ1/report.ipynb`
- PJ1 报告导出文件：`PJ1/report.pdf`、`PJ1/report.html`
